📖 Description
---
일상에서 꼭 필요한 날씨 예보, 외출하기 전, 창문 열기 전 확인했던 날씨 예보를 전용으로 한 사이트를 제작하였습니다. 

우산을 챙길지, 옷을 뭘 입어야할지 고민이 되신다면 날씨 예보 사이트에서 확인하세요!
